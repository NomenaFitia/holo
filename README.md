# holo

Option pour réduire les détails du maillage 
- garder les pixels centraux dans un groupe 3*3 , en commençant par le pixel d'origine 0 , 0
- prendre en compte les distances entre les pixels ( *2 dans le cans d'un groupe 3*3 car on enlève les pixels intermédiaires)
- système de leap pour sauter quelques fichiers en prenant en compte l'espacement entre les coupes
